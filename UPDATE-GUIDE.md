# 小亮网站更新指南

## 🎯 更新方式

你有两种方式更新小亮的网站：

### 方式一：通过我更新（推荐）⭐⭐⭐⭐⭐

**最简单的方式，你只需要发给我内容！**

**步骤：**

1. **发送内容给我**：
   - 📸 拍摄小亮的照片发给我
   - 📝 发送文字描述（今天发生了什么）
   - 🎬 如果有视频，也可以发给我

2. **我会自动处理**：
   - 整理文件到对应目录
   - 更新 `data/diary.json` 数据
   - 推送到 GitHub
   - 网站自动更新

**示例对话：**

```
你：[发送照片]
你：今天小亮学会了吹口哨！很可爱
我：好的！我来更新网站...
我：✅ 更新完成！网站已同步
```

### 方式二：手动更新

如果你想自己更新，可以按以下步骤操作：

#### 1. 更新数据文件 `data/diary.json`

打开 `data/diary.json` 文件，添加新的日记条目：

```json
{
  "stats": {
    "age": 5,
    "scales": 0,
    "trust": 1,
    "cute": 100
  },
  "diary": [
    {
      "date": "2026-03-18",
      "day": 1,
      "content": [
        "第一次见面，小亮的红色腮红很漂亮！",
        "对手机屏幕很好奇，会歪头观察",
        "喜欢站在栖木的最高处",
        "食欲正常，吃得很香",
        "建立信任关系的开始"
      ],
      "images": [],
      "video": null
    },
    {
      "date": "2026-03-19",
      "day": 2,
      "content": [
        "小亮今天学会了吹口哨！",
        "特别活泼可爱",
        "吃了最喜欢的苹果"
      ],
      "images": [
        "images/2026-03-19/001.jpg",
        "images/2026-03-19/002.jpg"
      ],
      "video": "https://www.bilibili.com/video/BVxxxxx"
    }
  ],
  "gallery": []
}
```

#### 2. 添加图片

将照片放到 `images/` 文件夹中，建议按日期组织：

```
images/
├── 2026-03-18/
│   ├── 001.jpg
│   └── 002.jpg
└── 2026-03-19/
    ├── 001.jpg
    └── 002.jpg
```

#### 3. 提交并推送

```bash
git add .
git commit -m "update: 添加2026-03-19的日记"
git push
```

#### 4. 等待自动部署

推送到 GitHub 后，Netlify 会自动部署，通常需要 1-2 分钟。

## 📊 数据结构说明

### 成长数据 (stats)

```json
{
  "age": 5,      // 月龄
  "scales": 0,   // 学会的音阶数量
  "trust": 1,    // 信任等级（1-5）
  "cute": 100    // 可爱指数（0-100）
}
```

### 日记条目 (diary)

每条日记包含：

- `date`: 日期（格式：YYYY-MM-DD）
- `day`: 第几天
- `content`: 日记内容（数组，多条记录）
- `images`: 图片路径数组
- `video`: 视频链接（可选）

## 🎬 视频嵌入

支持的视频平台：

1. **B站**：使用嵌入代码
   ```
   https://www.bilibili.com/video/BVxxxxx
   ```

2. **YouTube**：使用嵌入链接
   ```
   https://www.youtube.com/embed/视频ID
   ```

3. **本地视频**：放在 `videos/` 文件夹
   ```
   "video": "videos/2026-03-19/singing.mp4"
   ```

## 🆘 常见问题

### Q: 更新后网站没有变化？

A: 清除浏览器缓存或等待 2-3 分钟，Netlify 部署需要时间。

### Q: 如何查看网站访问数据？

A: 登录 Netlify 后台，在 "Analytics" 中查看访问统计。

### Q: 如何修改网站标题？

A: 编辑 `index.html` 文件中的 `<title>` 标签。
